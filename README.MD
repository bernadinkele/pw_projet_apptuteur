# Gestion des Tuteurs et Visites

Ce projet Symfony permet de gérer des tuteurs, leurs étudiants et les visites de suivi. Il inclut une interface web pour les tuteurs et une API REST via **API Platform** pour la création des tuteurs et des étudiants.

---

## Prérequis

Avant de commencer :  

- Docker Desktop doit être lancé  
- Vos conteneurs Symfony fonctionnent avec le **nouveau Dockerfile** fourni  

---
